Etiquetas HTML y atributos

En este ejercicio, repasaré la funcionalidad de algunas etiquetas HTML, sus caracteristicas y sus atributos

Button

La etiqueta button, crea un boton simple, por defecto rectangular con texto en su interior, permite la ejecucion de eventos como por ejemplo, enviar o borrar los datos de un formulario o muchas cosas mas si se combina con algun lenguaje de programacio como JavaScript

Es compatible (segun w3schools) con Chrome, Firefox, Edge, Safari y Opera. Ademas permite la introducción de etiquetas i, b, strong, br y img en su interior, cosa que no permite un boton creado con la etiqueta imput

Algunos de sus atributos son:

  • Autofocus. Este atributo permite que el foco se situe sobre este botón al cargar la pagina.
  • Disabled. Si este atributo se encuentra en la declaracion del botón, este se mostrará pero se encontrará inutilizado.
  • Type. Este atributo especifica el tipo de botón, normal (para ejecución de eventos), reset (para resetear todos los campos del formulario), y sumbit (para enviar los datos del formulario)
  • Form. Este atributo basico de las etiquetas de formulario, especifica a que id de formulario debe hacer referencia si el tipo del botón es sumbit.

Img (Origen de HTML)

La etiqueta img nos permite insertar imagenes en nuestras paginas web. Esta etiqueta requiere de 2 atributos, src y alt.

Es compatible (segun w3schools) con Chrome, Firefox, Edge, Safari y Opera. Permite la inserción en el interior de etiquetas a para así poder asignar un enlace a la imagen

Div (HTML 3.2)

La etiqueta div nos da la posibilidad de crear contenedores, los cuales pueden variar de tamaño o posición gracias a CSS.

Es compatible (segun w3schools) con Chrome, Firefox, Edge, Safari y Opera. Puede contener cualquier tipo de etiqueta en su interior.

Es compatible (segun w3schools) con Chrome, Firefox, Edge, Safari y Opera para los formatos de mp4 y WebM y Chrome,Firefox, Edge y Opera para el formato de ogg.
